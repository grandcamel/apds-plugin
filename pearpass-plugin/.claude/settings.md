# PearPass Plugin Development Settings

## Project Context

This is a Claude Code plugin for supporting developers working on PearPass Desktop.

## Development Guidelines

1. Skills should be invocable and provide clear instructions
2. Agents should have appropriate tool restrictions
3. All components should follow Claude Code plugin conventions
4. Security is paramount - this is a password manager plugin

## Testing

Test the plugin by:
1. Adding it to Claude Code settings
2. Using the skills in a PearPass development context
3. Verifying agents provide helpful guidance
